# 数据挖掘与机器学习分类算法可视化系统

本项目是一个基于前后端分离架构的分类算法教学与演示 Demo。系统旨在通过直观的 Web 界面，展示从经典数据挖掘到现代深度学习分类算法的核心原理、运行过程及评价指标。

系统主要包含以下六大经典分类算法的运行原理与可视化展示：

### 1. 传统统计与距离模型
- [ ] **K-近邻算法 (KNN)**
  - **核心原理**：基于空间距离度量的投票分类机制。
  - **可视化展示**：2D/3D 散点图中未知数据点分类过程的动态连线展示。
  - **评价指标**：不同 K 值对分类准确率 (Accuracy) 的影响曲线。
- [ ] **逻辑回归 (Logistic Regression)**
  - **核心原理**：基于 Sigmoid 函数的线性边界概率映射。
  - **可视化展示**：2D 空间中的平滑线性决策边界划分及 Sigmoid 曲线。
  - **评价指标**：对数损失 (Log Loss) 随迭代次数下降的折线图。
- [ ] **线性判别分析 (Linear Discriminant Analysis - LDA)**
  - **核心原理**：“类内方差最小，类间方差最大”的投影降维与分类思想。
  - **可视化展示**：高维数据点向最佳判别轴投影并最大化可分性的动态过程。
  - **评价指标**：混淆矩阵 (Confusion Matrix) 与特征空间分布图。

### 2. 树模型、集成学习与深度学习
- [ ] **决策树 (Decision Tree)**
  - **核心原理**：基于信息增益或基尼系数的特征空间层级划分。
  - **可视化展示**：直观的树状层级结构图及节点分裂规则演示。
  - **评价指标**：剪枝前后的模型复杂度与测试集准确率对比。
- [ ] **AdaBoost 集成学习**
  - **核心原理**：通过前向分布算法将多个弱分类器动态加权聚合成强分类器。
  - **可视化展示**：每一轮迭代中错分样本权重 (Sample Weights) 放大的视觉变化追踪。
  - **评价指标**：基分类器数量对最终集成模型性能提升的评估。
- [ ] **多层感知机 (Multilayer Perceptron - MLP)**
  - **核心原理**：基于反向传播算法和激活函数的前馈神经网络模型。
  - **可视化展示**：神经网络拓扑结构图，以及隐藏层如何扭曲空间生成复杂的非线性决策边界。
  - **评价指标**：训练轮数 (Epochs) 与训练集/验证集 Loss 及准确率的实时变化曲线。

## 🛠️ 技术栈选型

- **前端 (Frontend)**: Vue 3 + Vite + ECharts (用于高质量数据图表与边界动态渲染)
- **后端 (Backend)**: FastAPI (提供轻量、高性能的异步 RESTful API)
- **核心算法库**: 
  - `Scikit-learn` (封装核心算法与模型评估工具)
  - `NumPy` & `Pandas` (用于高效矩阵运算与数据集预处理)

## 📁 目录结构说明

- `/frontend`: Vue 3 前端工程代码
- `/backend`: FastAPI 后端工程代码
  - `/api`: RESTful 路由与接口控制器
  - `/core`: 算法模型封装与核心运算逻辑
- `/dataset`: 存放系统运行所需的测试数据集 (如 CSV 文件)
